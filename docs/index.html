MeMesh Plugin

Searchable project memory for Claude Code

npm install -g @pcircle/memesh click to copy

I love Claude Code. It changed how I build software.

I built this plugin to help more people get the most out of Claude Code for vibe coding. When projects grow, it gets hard to track decisions across sessions. MeMesh remembers for you.

What it does

🔍

Search by meaning

Find past decisions by what you mean, not exact keywords

📝

Auto-tracking

Decisions and patterns saved automatically as you work

🔒

100% local

All data stays on your machine. Nothing uploaded

⚡

Zero config

Install and restart Claude Code. That's it

3 commands, that's all

buddy-do "task" Run a task with memory context
buddy-remember "topic" Search past decisions and context
buddy-help Show available commands
